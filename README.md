# Time and Frequency Domain Analysis of Signals using MATLAB

This project demonstrates analysis of signals in time and frequency domains using MATLAB.

## Concepts Covered
- Time-domain signal representation
- Linear and circular convolution
- Frequency spectrum analysis using FFT
- Sampling effects

## Tools Used
- MATLAB
- Signal Processing Toolbox
